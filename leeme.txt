CARPETA RAIZ DEL PROYECTO
- CREAR EL ARCHIVO PACKAGE.JSON (MUY IMPORTANTE, LO PRIMERO QUE HAY QUE HACER):
	TERMINAL,"npm init".
	ESTO NOS CREARÁ EL PACKAGE.JSON, EL PACKAGE_LOCK.JSON Y LA CARPETA NODE_MODULES (QUE ES
	DONDE SE ENCUENTRAN TODAS LAS DEPENDENCIAS QUE USAREMOS EN EL PROYECTO CON SUS VERSIONES).
	NOTA: PODEMOS BORRAR NODE_MODULES (PARA ENVIARLO POR MAIL POR EJEMPLO PORQUE PESA MUCHO) Y RECUPERARLO
	SIRMPRE QUE TENGAMOS EL PACKAGE.JSON Y HACIENDO "npm install".
- INSTALAR SASS:
	TERMINAL, "npm install sass --save-dev".
- INSTALAR GULP:
	TERMINAL, "npm install gulp --save-dev".
- INSTALAR GULP-SASS(para poder compilar el sass a través de gulp):
	TERMINAL, "npm install gulp-sass --save-dev".
- INSTALAR GULP-WEBP(para transformar el jpg y el png en formato webp):
	TERMINAL, "npm install gulp-webp --save-dev".
- INSTALAR (estas dependencias mejoran el velocidad de las web):
	TERMINAL, "npm install gulp-imagemin@7.1.0 --save-dev".
	TERMINAL, "npm install gulp-cache --save-dev".
- INSTALAR (estas dependencias mejoran el código css al finalizar el proyecto):
	TERMINAL, "npm install cssnano --save-dev" (comprime el código css).
	TERMINAL, "npm install autoprefixer --save-dev" (se encarga que el proyecto funcione bien en todos los navegadores).
	TERMINAL, "npm install postcss --save-dev" (hace algunas tranformaciones por medio de los dos anteriores).
	TERMINAL, "npm install gulp-postcss --save-dev" (idem de arriba).
- INSTALAR (mejora el código js):
	TERMINAL, "npm install gulp-terser-js --save-dev".
- INSTALAR (hace que podamos encontrar posibles errores en el scss dentro del inspector de elementos del navegador,
suponiendo que tenemos todo el código css comprimido, ya que sería imposible):
	TERMINAL, "npm install gulp-sourcemaps --save-dev".

NOTA IMPORTANTE: TODAS ESTAS DEPENDENCIAS DE DESARROLLO PODEMOS INSTALARLAS DE UNA VEZ HACIENDO:
"npm install sass gulp gulp-sass gulp-webp gulp-imagemin@7.1.0 gulp-cache cssnano autoprefixer postcss gulp-postcss gulp-terser-js gulp-sourcemaps --save-dev"

NOTA IMPORTANTE: el archivo principal es un ".html", hay una carpeta "includes" que dentro trae archivos ".php"...en caso que queramos usarla debemos cambiar la extensión del princpipal a ".php".
